# Public Real Document Artifacts

Generated by `scripts/run_public_real_cases.py` from `examples/public_real_documents/manifest.json`.
Each case contains the official input copy, source metadata, human labels, result.json, trace.json, summary.md, and retrieval exports.

Scope: these are official public documents used for lightweight external-generalization evidence. The labels check key facts and text evidence, not full OCR character accuracy or table-cell accuracy. Long PDFs may declare a `page_range` in source metadata when the online MinerU Agent API page limit applies.
