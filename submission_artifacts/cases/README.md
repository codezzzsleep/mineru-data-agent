# Submission Case Artifacts

These artifacts are generated by scripts/run_submission_cases.ps1 from examples/batch_manifest_5cases.json.
Each case directory contains the input fixture, result.json, trace.json, summary.md, and retrieval export files.

Important boundary: these five cases are self-constructed HTML/web fixtures. They are used to verify the Data Agent pipeline, validator, recovery logic, trace writing, and retrieval export end to end. Their quality scores are rule-based "no blocking issue detected" scores, not field-level/table-level accuracy on real PDFs. Use `submission_artifacts/mineru_cases/` for MinerU CLI PDF evidence.
