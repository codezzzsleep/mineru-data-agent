# Challenge Case Fixtures

These are synthetic but more realistic challenge fixtures for the submission.
They are designed to stress review points that the base five HTML cases do not
fully cover: cross-page-style totals, OCR noise, standards matrices, and
incident workflow recovery evidence.

They are not customer documents and should not be described as external real
data. Human labels are stored in `examples/evaluation/labels.json` and summarized
in `submission_artifacts/challenge_cases/human_annotation_table.md`.
